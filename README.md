# Babu Shobhnath Singh Memorial Trust

A bilingual (Hindi/English) Progressive Web Application for the Babu Shobhnath Singh Memorial Trust.

## Features
- Bilingual Support (Hindi/English)
- Guest Browsing
- User Authentication via Replit Auth
- Content Management (Events, Blogs, Poetry)
- Admin Portal with Role-based Access
- Media Uploads
- PWA Ready

## Tech Stack
- Frontend: React 18, TypeScript, TailwindCSS, Shadcn UI
- Backend: Express.js, PostgreSQL, Drizzle ORM
- Auth: Replit Auth (OIDC)
- Logging: Winston

Built with Replit Agent.
